请解释一下什么是 JavaScript 的闭包，以及它在实际开发中的应用场景？

已完成
用时：5分32秒

闭包是指有权访问另一个函数作用域中变量的函数。创建闭包的常见方式是在一个函数内部创建另一个函数。闭包的应用场景包括：数据私有化、函数柯里化、模块化开发等。

什么是 CSS 盒模型？标准盒模型和 IE 盒模型有什么区别？

已完成
用时：3分48秒

CSS 盒模型包括内容区、内边距、边框和外边距。标准盒模型的 width/height 只包含内容区，IE 盒模型的 width/height 包含内容区、内边距和边框。可以通过 box-sizing 属性切换。
